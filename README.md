# Cache Api

## Getting Started

Simple rest api built with Node.js, Express.js and MongoDB.

### Prerequisites

You need Node.js, Npm.js and MongoDB installed to run the project.

## Built With

*  Node v12.16.3
*  Npm v6.14.4 
*  MongoDB v4.2.6
